# 审查发现 修复方案 落地模块
1 B3≈B4 时三种解释不可区分（contamination） Naive LLM 无数据直接提问 → 若答对则标记 contamination；至少 1 个非公开 GT case contamination.py
2 Safety 非 gate 函数，幻觉 Agent 可被"洗白" Safety < 0.7 → 总分 ×0.5 惩罚因子，标记 "NOT TRUSTWORTHY" metrics.py
3 T3-T5 GT 循环验证风险 HR tolerance ±0.15；GT 声明为"反映 ITIP 特定管线"；3 个关键基因独立交叉核对 tasks.py
4 硬规则误杀方法学引用 区分 evidence/method citations；维护 20+ 篇方法学论文白名单 hallucination.py
5 跨 task 线性归一化引入分布偏差 Z-score 归一化 (x-μ)/σ；同时报告 raw + normalized；雷达图替代单一 Overall reporter.py
6 Test case 非独立性未建模 Bootstrap CI (gene-level resampling)；effective n < 5 时标记 exploratory runner.py
7 T1-LIT 高引=旧论文 时间分层：GT 强制 ≥20% 近 3 年论文（不设引用门槛）；per-year-group Recall@K tasks.py
8 T2-GDA 双源矛盾无仲裁 三级置信度：双源一致=high / 单一来源=moderate / 分别报告 Accuracy tasks.py
9 P0-4 验证 n=1 无统计意义 ≥10 个注入样本（5真+5假混合），recall ≥0.8 & precision ≥0.9 hallucination.py
10 人工评分单人无信度 4/12 份双评，报告 Cohen's κ；κ < 0.6 则降级为 preliminary scorer.py
11 多重比较未校正 预注册 primary hypotheses；其余 exploratory + BH 校正 runner.py
12 Completion Rate 可被刷 区分 (a)有答案完成 / (b)合理拒绝(同分) / (c)崩溃超时 metrics.py
