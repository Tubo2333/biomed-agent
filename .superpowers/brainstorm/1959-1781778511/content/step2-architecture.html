模块全景图（共 8 个代码文件 + data/）

biomed-agent/
├── src/
│   ├── benchmark/
│   │   ├── types.py          # BenchmarkTask / AgentEvalMetrics / EvalAgent Protocol
│   │   ├── tasks.py          # 5 个 Task 定义 + Ground Truth 加载器
│   │   ├── metrics.py         # 4 维 Metrics 计算引擎
│   │   ├── runner.py          # BiomedBenchmark: 遍历 Task×Agent → 产报告
│   │   ├── hallucination.py    # 硬规则 + 软分级 幻觉检测器
│   │   ├── baselines.py       # B1/B2/B3/B4 Agent 实现
│   │   ├── scorer.py          # 人工评分模板 + 自动评分协调
│   │   └── reporter.py         # 结果聚合 + JSON/CSV 导出
├── data/
│   └── benchmark/
│       └── ground_truth/     # 每个 Task 的 gt JSON 文件
└── results/
    ├── benchmark_v1.json     # 完整结果
    └── benchmark_comparison.csv
